## 1. GENERAL FORMATTING RULES
- **No Preamble:** Do not include `\documentclass` or `\begin{document}`. Output only the content within the `ex` environments.
- **Math Mode:** All numerical values, variables, formulas, and units must be enclosed in LaTeX math mode `${ ... }$`, and MUST use `{` and `}` inside `$` (e.g. ${1}$).
- **Text Integrity:** Do not summarize, shorten, or alter the original content. If a word is misspelled or an English term is localized incorrectly, correct it to the standard professional term.
- **Decimals:** Use `{,}` for decimal points in math mode to ensure proper spacing (e.g., `$1{,}5$`).
- **Brackets:** Use `\left(` and `\right)`, `\left[` and `\right]`, etc., to ensure brackets automatically scale with the content (especially for fractions).
- **Quotes:** Use standard LaTeX double quotes: `` `` (double backtick for opening, '' double apostrophe for closing).
- **Physical Units:** Use `\mathrm{...}` for units (e.g., `${10 \, \mathrm{m/s^2}}$`).
- **Vectors:** Use `\overrightarrow{...}`.
- **Terminology Standardization:** If names of vectors, scientists, physical constants, units, or technical terms are incorrectly localized, misspelled, or written in non-standard Vietnamese forms, convert them to their internationally accepted standard forms (e.g., Newton, Coulomb, Joule, Ampere, Ohm, Vector). Do not keep incorrect or informal variants.
- **FORBIDDEN COMMANDS:** DO NOT use `\displaystyle`. DO NOT use spacing commands like `\quad`, `\qquad`, `\hfil`, `\hfill`, `\vspace`, `\hspace`.
- For simultaneous equations (AND), use: `\left\{\begin{aligned} #1 \end{aligned}\right.`.
  For example: `${\left\{\begin{aligned} & a = 1 \\ & b = 2 \end{aligned}\right.}$`
- For alternative conditions (OR), use: `\left[\begin{aligned} #1 \end{aligned}\right.`.
- Do not use the `cases` environment.
- For integrals, always use `\int\limits_{lower}^{upper}` to ensure limits are placed vertically.
- Absolutely NO periods/dots (.) at the end of any option within `\choice` or `\choiceTF` commands. Example: use `\choice{Đáp án A}{Đáp án B}{...}{...}` INSTEAD OF `\choice{Đáp án A.}{...}`.
- Some explanations contain a line such as "Chọn X", "Đúng", "Sai", "Hướng dẫn giải", "(Group Vật lý Physics)", etc. When writing the solution/explanation, DO NOT include or repeat this line. (You MAY still use such a marking to decide which option is `\True`, but do not copy the line into the output.)

## 2. TABLES — VARIATION / SIGN TABLES (tkz-tab)
- Do NOT use `tabular` for variation tables.
- All variation tables (bảng biến thiên) or sign tables (bảng xét dấu) MUST be written using the `tkz-tab` package with this format:
`\begin{tikzpicture}
    \tkzTab
    {$x$/0.8, $f'(x)$/0.8, $f(x)$/3}
    {$-\infty$, $-1$, $1$, $3$, $+\infty$}
    {,+,0,-,d,-,0,+,}
    { -/ $-\infty$, +/ $-4$, -D+ / $-\infty$ / $+\infty$, -/ $4$, +/$+\infty$}
\end{tikzpicture}`
- 'd' represents a double vertical line (undefined point).
- 'D+' or 'D-' represents limits at points of discontinuity.

## 3. FIGURES, GRAPHS AND IMAGES — USE `\includegraphics`, DO NOT DRAW WITH TikZ
- **DO NOT recreate figures, function graphs, geometric shapes, circuit diagrams or photos using TikZ.** (The only TikZ you may produce is the `tkz-tab` variation/sign table above.)
- Whenever a question or its solution contains any figure/graph/diagram/photo, insert it as an image using `\includegraphics`, taken from the provided **Asset Manifest** (the cropped figure images supplied after the full page images).
- **Anchoring:** match each figure's vertical position with the corresponding question text, and place the `\includegraphics` at that spot in the question.
- **REFERENCING:** Use the provided "Asset Manifest" to obtain the correct image filename. Example filename: `fig_p0_1_scale0.2.png`. Reference it exactly by that filename inside the braces: `\includegraphics[width=0.2\linewidth]{fig_p0_1_scale0.2.png}`.
- **SCALE RULE:** the `<scale>` in `width=<scale>\linewidth` MUST equal the number after `scale` in the filename. e.g. `fig_p0_1_scale0.2.png` -> `width=0.2\linewidth`.
- **PLACEMENT:**
   - Use `\begin{center}\includegraphics[width=<scale>\linewidth]{...}\end{center}` for a centered image.
   - Use `\immini[thm]{...question text...}{\includegraphics[width=<scale>\linewidth]{...}}` if the image should sit next to the text (MUST use `thm`).
- If a figure clearly belongs to a question but NO matching cropped image exists in the Asset Manifest, simply omit the figure (do NOT write a placeholder, do NOT invent TikZ).

## 4. SPECIFIC SYNTAX STRUCTURES

### A. Multiple Choice (Trắc nghiệm)
- Use the `\choice` command with 4 sets of curly braces.
- Mark the correct answer with `\True` (only if the source indicates it; see section 0).
- Structure:
\begin{ex}
    [Question Content]
    \choice
    {Option A}
    {\True Option B}
    {Option C}
    {Option D}
    \loigiai{[Solution copied from source, or empty]}
\end{ex}

### B. True/False (Đúng sai)
- Use the `\choiceTF` command. Mark correct statements with `\True` (only if the source indicates it).
- In the `\loigiai` section, you MUST use exactly 4 `\itemch` commands corresponding to the 4 statements.
- If a statement has no explanation in the source, leave the space after `\itemch` empty.
- MUST use `[1]` in `\choiceTF[1]`.
- Structure:
\begin{ex}
    [Question Content]
    \choiceTF[1]
    {\True Statement 1}
    {Statement 2}
    {\True Statement 3}
    {\True Statement 4}
    \loigiai{
        \begin{itemchoice}
        \itemch
        \itemch Explanation for statement 2 (only if present in source)
        \itemch
        \itemch Explanation for statement 4 (only if present in source)
        \end{itemchoice}
    }
\end{ex}

### C. Short Answer (Trả lời ngắn)
- Structure:
\begin{ex}
    [Question Content]
    \shortans{Result}
    \loigiai{[Solution copied from source, or empty]}
\end{ex}

### D. Essay/Open-Ended (Tự luận)
- Structure:
\begin{ex}
    [Question Content]
    \loigiai{[Solution copied from source, or empty]}
\end{ex}

- Variant with an image next to the question content: MUST use `thm`. Any image inside `\loigiai{}` MUST be centered using `\begin{center}...\end{center}`.
\begin{ex}
\immini[thm]{
        [Question Content]
    }{
        [Image code if any]
    }
    \choice
    {Option A}
    {\True Option B}
    {Option C}
    {Option D}
    \loigiai{[Solution copied from source, or empty]}
\end{ex}

### E. Multi-part Questions (Shared Stimulus)
- Use `\sochc{number_of_parts}{stimulus_content}` (e.g. when the source says `Sử dụng các thông tin sau cho Câu XX và Câu XX:`).
- ALWAYS use `thm` inside `\sochc{number_of_parts}{stimulus_content}` when using `\immini` (i.e. `\immini[thm]`).
- Structure:
\begin{ex}
\sochc{2}{\immini[thm]{
        [Common stimulus text]
    }{
        [Image code if any]
    }
}
    \begin{chc}
        [Sub-question 1]
        \loigiai{[Solution 1, or empty]}
    \end{chc}
    \begin{chc}
        [Sub-question 2]
        \loigiai{[Solution 2, or empty]}
    \end{chc}
\end{ex}
